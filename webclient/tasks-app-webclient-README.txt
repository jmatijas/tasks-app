tasks-app weblicent is based on Material UI free template: Devias Kit Admin Dashboard

Devias Kit Admin Dashboard cloned from GitHub and copied to the tasks-app/webclient.

It is create-react-app project.

Quick start
Download from Github or Download from Devias or clone the repo: git clone https://github.com/devias-io/react-material-dashboard.git

Make sure your NodeJS and npm versions are up to date for React 16.8.6


!!! INSTALL !!! dependencies:   npm install or yarn


!!! START !!! the server:  npm run start or yarn start


Views are on: localhost:3000
